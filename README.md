# Tarish

XMRig wrapper for easy mining with auto CPU detection.

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/0x0FFF0/tarish/main/install.sh | sudo bash
```

## Usage

```bash
sudo tarish install      # Install to system
tarish start             # Start mining
tarish stop              # Stop mining
tarish status            # Check status
sudo tarish service enable   # Auto-start on boot
```

## Uninstall

```bash
sudo tarish uninstall
```
